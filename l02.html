Crime Mapping and Spatial Analysis

Lab 2: GIS Mapping

John Palmer

Task 1

    Go to the ICGC website and select a 1:5.000 orthophoto that covers our building and the streets immediately around it. Download this as a TIFF (GeoTiff) file. (You will need to enter your email address and wait for the confirmation email to do the download.)

    Open QGIS and start a new project. Add your orthophoto to the project using the menu items Layer → Add Layer → Add Raster Layer. (See the Add Raster recipe.)

Task 2

    Go downstairs and spend a few minutes observing the sidewalk along each side of this building. Write down the approximate number of people on each sidewalk.

Task 3

    On your QGIS map, create a new polygon layer. Give the layer attributes that include whole number (integer) fields not only for ID, but also for number of people observed (label this field N). Draw a rectangle over each of the 4 stretches of sidewalk you observed, giving each rectangle a sequential ID starting with 1, and adding the number of people you observed in the N field. (See the Creating New Vector Layer and Adding Features recipes.)

Task 4

    Use the layer properties to adjust your new polygon layer such that it is visible without entirely blocking the underlying orthophoto. Add labels that show the number of people you counted on each section of sidewalk. (See the Adjusting Vector Layer Appearance recipe.)

Task 5

    Download the OpenStreetMap vector for this same area. Based on the Adding OpenStreetMap vectors recipe, add lines showing the tertiary streets (which include Carrer de Ramon Trias Fargas). Adjust the style of these lines so that they are easy to see on the map.

Lab Report (turn in via Aula Global before Thursday lecture)

  • Use the Print Composer to compose and export a map that includes the information you added in each of the tasks. Export the map as a PDF file and turn it in.
